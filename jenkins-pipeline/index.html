发表于 2019-03-18 | 分类于 jenkins pipeline | | 围观次数:

由于项目是在gitlab上面，一个项目很多同事都需要发布，而且分支肯定是不一样的，这就需要支持用户选择自己的分支去发布。这篇文章就是介绍如何配置和支持选择分支发布的.
